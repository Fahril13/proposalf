# 🏖️ Surprise Beach Trip Proposal for Freya

Sebuah *mini web-app* interaktif yang dibuat khusus sebagai proposal meminta izin untuk mengajak Freya liburan singkat ke pantai setelah masa ujian selesai. 

## ✨ Fitur
* **Desain Interaktif & Responsif:** Nyaman dibuka lewat HP dengan format *swipeable slides* (bisa di-geser seperti *story*).
* **Informasi Lengkap:** Berisi alasan, opsi destinasi, protokol keamanan, hingga *itinerary* (rundown) perjalanan.
* **Tautan Langsung ke Maps:** Tombol yang mengarah langsung ke titik lokasi di Google Maps.

## 🛠️ Teknologi yang Digunakan
* HTML5
* CSS3 (Custom properties, Flexbox, CSS Grid)
* Vanilla JavaScript (Logic navigasi & deteksi *swipe* di *mobile*)
